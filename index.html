Introduction & Problem Statement

Purpose: The goal is to develop an autonomous security robot for real-time monitoring and person detection in various environments.

Problem: Traditional security systems have limitations including static positioning, high energy consumption, and lack of intelligent analysis. This project provides an automated, continuous monitoring solution that addresses these issues through mobility, solar power, and AI-powered detection.

System Architecture & Components

Hardware

  • Raspberry Pi Pico & Pi 3: Dual-processor setup for efficient task distribution
  • Sensors: AI Camera (for person detection), IR (for line tracking), Ultrasonic (for obstacle avoidance), Gyroscope (for stability)
  • Solar-powered battery system: 18650 battery with solar charging capability for extended operation

Software

  • Blynk IoT Platform: For remote control and monitoring
  • Gmail API: For instant alert notifications
  • Custom AI Model: For person detection using TensorFlow Lite

Key Features & Functions

Raspberry Pi 3

Acts as the main processing unit for the robot, managing data from sensors and controlling operations with its quad-core processor.

Raspberry Pi Pico

Functions as a compact controller for sensor integration and movement commands with its RP2040 microcontroller.

Raspberry Pi AI Camera

Used for real-time video surveillance and human detection with 8MP resolution and 1080p video.

L298N Motor Driver

Controls the motors that facilitate movement of the robot with dual H-bridge design.

TT DC Motors

Provides the movement capability for the robot with 200RPM speed and 1:48 gear ratio.

5-Channel IR Sensor

Used for detecting the robot's path and tracking its navigation with 5 independent sensors.

MPU6050 Gyroscope Sensor

Assists in maintaining stability and orientation of the robot with 6-axis motion tracking.

Ultrasonic Sensor

Measures distance and detects obstacles in the robot's path with 2cm-400cm range.

NEO6MV2 GPS Module

Provides global positioning data to help with navigation with 2.5m position accuracy.

Light Dependant Resistor (LDR)

Senses ambient light levels to assist in navigation adjustments and solar charging optimization.

18650 Battery

Supplies power to the robot's systems (3.7V, 2600mAh), rechargeable via solar panels.

HC-05 Bluetooth Module

Enables wireless communication for remote control and monitoring with 10m range.

Buzzer

Provides audio alerts (85dB) for security notifications and system status.

LED Indicators

Used to signal different operational statuses of the robot with RGB capabilities.

Custom 3D Printed Casing

Houses the components securely and protects them from environmental factors with PLA material.

Results & Performance

  • Object Detection Accuracy: Achieved 92% accuracy in person detection during daylight conditions
  • Alert Response Time: Average 3.2 seconds from detection to notification
  • Solar Energy Efficiency: 6 hours of continuous operation on full battery with solar charging maintaining charge in daylight
  • Line Tracking Precision: 95% accuracy in following pre-defined paths
  • Obstacle Avoidance: Successfully detects and navigates around obstacles up to 30cm away

Applications

  • Home and Industrial Surveillance: Autonomous patrols in residential areas, warehouses, and factories
  • Smart Home Monitoring: Integration with smart home systems for comprehensive security
  • Campus Security: Monitoring university or school premises during off-hours
  • Construction Site Monitoring: Night-time surveillance of equipment and materials
  • Agricultural Monitoring: Protecting farm equipment and monitoring livestock areas

Conclusion & Future Improvements

This solar-powered surveillance robot provides a smart, sustainable solution for automated security monitoring, combining mobility, renewable energy, and AI-powered detection in a single platform.

Future Enhancements:

  • Advanced AI: Implement facial recognition and behavior analysis for more sophisticated detection
  • Low-light Camera: Upgrade to starlight camera for better night-time performance
  • Weatherproofing: IP65-rated casing for outdoor operation in all conditions
  • Wireless Charging: Implement induction charging stations for automatic recharging
  • Swarm Technology: Enable multiple robots to work together for large area coverage
