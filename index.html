Introduction & Problem Statement

Purpose: The goal is to develop an autonomous security robot for real-time monitoring and person detection.

Problem: Traditional security systems have limitations. This project provides an automated, continuous monitoring solution.

System Architecture & Components

Hardware

  • Raspberry Pi Pico & Pi 3
  • Sensors: AI Camera, IR, Ultrasonic, Gyroscope
  • Solar-powered battery system

Software

  • Blynk (remote control)
  • Gmail (alert system)

Key Features & Functions

Raspberry Pi 3

Acts as the main processing unit for the robot, managing data from sensors and controlling operations.

Raspberry Pi Pico

Functions as a compact controller for sensor integration and movement commands.

Raspberry Pi AI Camera

Used for real-time video surveillance and human detection.

L298N Motor Driver

Controls the motors that facilitate movement of the robot.

TT DC Motors

Provides the movement capability for the robot.

5-Channel IR Sensor

Used for detecting the robot's path and tracking its navigation.

MPU6050 Gyroscope Sensor

Assists in maintaining stability and orientation of the robot.

Ultrasonic Sensor

Measures distance and detects obstacles in the robot‚Äôs path.

NEO6MV2 GPS Module

Provides global positioning data to help with navigation.

Light Dependant Resistor (LDR)

Senses ambient light levels to assist in navigation adjustments.

18650 Battery

Supplies power to the robot's systems, rechargeable via solar panels.

HC-05 Bluetooth Module

Enables wireless communication for remote control and monitoring.

Buzzer

Provides audio alerts for security notifications.

LED Indicators

Used to signal different operational statuses of the robot.

Custom 3D Printed Casing

Houses the components securely and protects them from environmental factors.
